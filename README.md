# MindMeal - Smart Recipe & Meal Management System

MindMeal is a full-stack web application designed for users to discover, manage, and favorite recipes with a seamless user experience. The project emphasizes modern UI/UX principles and clean architecture.

## Live Demo

The first demo version is now live and publicly accessible: https://mindmeal.ibahars.com

The frontend is deployed on Cloudflare Pages, and the backend runs in Docker containers on a Linux server behind a Caddy reverse proxy with automatic HTTPS.

## 🚧 Work in Progress (WIP)

The first demo is live, but the project is under active development. 

## Tech Stack

### Frontend
- **React.js (Vite):** Component-based UI architecture with a fast build pipeline.
- **Tailwind CSS:** Utility-first styling for custom themes and glassmorphism effects.
- **Lucide React:** Consistent and modern iconography.
- **React Router:** SPA navigation management.
- **Axios:** Asynchronous API communication.

### Backend
- **.NET 8 Web API:** High-performance RESTful services.
- **Entity Framework Core:** Object-Relational Mapping (ORM) for database management.
- **PostgreSQL:** Robust, lightweight relational data storage.
- **JWT Authentication:** Secure user authorization and session management with refresh tokens.

### Infrastructure & Cloud
- **Docker & Docker Compose:** Containerized API and database for consistent, reproducible environments.
- **Caddy:** Reverse proxy with automatic HTTPS (Let's Encrypt).
- **Cloudflare:** DNS, CDN, and Cloudflare Pages hosting for the frontend.
- **Cloudinary:** Cloud-based image storage and on-the-fly transformations.
