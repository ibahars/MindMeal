# MindMeal - Smart Recipe & Meal Management System

MindMeal is a full-stack web application designed for users to discover, manage, and favorite recipes with a seamless user experience. The project emphasizes modern UI/UX principles, clean architecture.

## 🚧 Work in Progress (WIP)

The project is actively being refactored to align with industry-standard software architectures. 

## Features

### Recipe Discovery & Management
* **Recipe Gallery:** A responsive grid display of recipes with dynamic badges for new items and categories.
* **Category Tagging:** Specialized categorization for Breakfast, Main Course, Dessert, Drinks, Snacks, and Healthy Living.
* **Detailed Instructions:** A dedicated view for recipe steps featuring high-quality hero images with optimized focal points and accessibility-focused typography.
* **CRUD Operations:** Authorized users can create, update, and delete recipes via a streamlined modal interface.
  
## Tech Stack

### Frontend
* **React.js:** Component-based UI architecture.
* **Tailwind CSS:** Utility-first styling for custom themes and glassmorphism effects.
* **Lucide React:** Consistent and modern iconography.
* **React Router:** SPA navigation management.
* **Axios:** Asynchronous API communication.

### Backend
* **.NET 8 Web API:** High-performance RESTful services.
* **Entity Framework Core:** Object-Relational Mapping (ORM) for SQL database management.
* **SQL Server:** Robust relational data storage.
* **JWT Authentication:** Secure user authorization and session management.
